# OVERVIEW

This repository contains two examples from my german Schnupperkursus.
It demostrates a simple quiz app in two versions: command line and WPF.
Even without my course, you can use this repository to learn C#.


# Überblick

Diese Projektmappe enthält Bespielprogramme für meinem Einsteiger-Schnupperkursus.
- QuizApp Kommandozeilen-Version
- QuizApp WPF-Version
